# För Evigt Smycken

## Beskrivning
För Evigt Smycken är en fiktiv e-handelswebbplats för bröllopssmycken skapad som slutuppgift i kursen HTML och CSS. Webbplatsen presenterar ett koncept för en webshop med fokus på smycken för bröllop och speciella tillfällen.

## Målgrupp
Målgruppen är personer som planerar bröllop eller söker stilrena smycken till annat.

## Struktur
Webbplatsen består av fyra sidor:

- index.html (startsida)
- produkter.html
- om-oss.html
- kontakt.html

## Tekniker som används
- HTML
- CSS
- Flexbox
- Grid
- Responsiv design

## Funktioner
- Responsiv layout för mobil, surfplatta och desktop
- Navigationsmeny med hamburgermeny i mobilvy
- Produktgrid för att visa smycken
- Kontaktformulär
- Tillgängliga bilder med alt-text
- SEO anpassad sajt, med korrekta h1, h2,h3, meta titlar och beskrivningar
