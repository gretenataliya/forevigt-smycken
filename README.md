# För Evigt Smycken

## Beskrivning
För Evigt Smycken är en e-handelswebbplats för bröllopssmycken och webbplatsen presenterar ett koncept för en webshop med fokus på eleganta och stilrena smycken för bröllop och högtidliga tillfällen.

## Målgrupp
Målgruppen är kvinnor som planerar bröllop eller söker eleganta smycken till festliga tillfällen.

Webbplatsen består av fyra sidor:
- index.html
- produkter.html
- om-oss.html
- kontakt.html

Semantiska HTML-element används:
header, nav, main, section, article och footer.

### Layout
Flexbox används för navigation och layout.
CSS Grid används för produktsektionen.

### Responsivitet
Mobile-first design används med brytpunkter vid:
480px och 768px.

### Tillgänglighet
Alla bilder har alt-text och formulär har labels.

### Formulär
Kontaktformulär finns på sidan kontakt.html.

### Mappstruktur

/assets
/images
/icons
/css
style.css

### Publicering
Sidan kan öppnas lokalt genom att öppna index.html i en webbläsare.
